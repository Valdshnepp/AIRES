О нас

Мы - команда молодых учёных, специализирующаяся в решении задач с применением современных информационных технологий и искусственного интеллекта.

Имеем профессиональный опыт в разработке приложений (web, mobile, desktop), обучении, внедрении и поддержке моделей искусственного интеллекта, применении методов компьютерного моделирования и интеллектуального анализа данных.

Предлагаем сотрудничество с целью развития, оптимизации и автоматизации бизнес-процессов для выхода вашей компании на новый техно-функциональный уровень.

Наши исследования

Робот-манипулятор нового поколения

Умная система автоматической сортировки и калибровки овощей по точным параметрам качества с использованием компьютерного зрения и машинного обучения.

Анализ сейсмических аномалий

Алгоритм машинного обучения для анализа сейсмических аномалий и оценки состояния аварийных зданий после землетрясений и других экстремальных воздействий.

Выявление онкологических заболеваний

Алгоритм машинного обучения для выявления онкологических заболеваний на основе анализа медицинских изображений — рентгеновских снимков и гистологических срезов.

Исследование тетраэдральной конфигурации спутников

Исследовательский проект по изучению эволюции тетраэдральной конфигурации из четырёх спутников, движущихся по кеплеровым орбитам для выявления гравитационных аномалий и следов новой физики.

Предсказание погодных условий

Проект по разработке сетевых моделей для предсказания погодных условий на основе климатических данных с использованием машинного обучения.

Криптовалютная система

Проект по созданию и развитию криптовалютной системы, основанной на современных принципах децентрализации и безопасности с быстрыми транзакциями.

Нейросетевая модель для детекции объектов

Продвинутая нейросетевая модель для детекции и отслеживания объектов в реальном времени с высокой точностью и скоростью обработки.

Робот сварщик

Автоматизированная система сварки с точным контролем траектории, параметров дуги и качеством шва для промышленного применения.
